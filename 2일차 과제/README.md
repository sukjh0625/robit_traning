외부 라이브러리 사용 보고서 (미사일)

주가 관련된 엑셀 파일 만드는 외부 라이브러리를 사용해서 코드를 짜본 후 다른 라이브러리도 사용해보고 싶어서 진로와 관련된 코드를 짜봄

실행방법 및 코드 설명: 실행 방법은 미사일 자체 무게, 탑재할 고체 연료의 무게, 발사 각도, 방위각, 초기 사출 속도를 적으면 고정된 중력, 대기밀도, 항력, 가중치를 이용해서 궤도를 계산해준다.
그 후 계산된 궤도를 그림으로 보여준다. 코드 설명은 거의 한줄 씩 코드에 주석으로 달아놓았다

라이브러리 이해: 총 3가지의 외부라이브러리를 사용해보았다
1. numpy
   수업시간에 배운 라이브러리로 연산을 빠르게 처리할 수 있게 해주는 라이브러리이다.
   (np.cos),(np.radians),(np.sqrt)같은 코드를 사용해서 백터연산, 라디안 변환등을 하는데 사용해보았다
2. scipy
   numpy기반 라이브러리 이지만 미분방정식 해결에 특화된 라이브러리이다.
   (solve_ivp)를 이용해 미분방정식을 통해 공기저항과 질량 변화를 고려하여 비행경로를 예측하였다
   또 땅에 닿는 순간을 찾고 멈추는 역할을 수행하였다
3. matplotilb
   그래프를 그려주는 라이브러리이다.
   3d 모형을 보여주는데 쓰였고, 경로들을 색깔로 구분하거나 점선으로 표현하는데 쓰였다.
   또한 돌려볼수도 있게 만들었다

외부 라이브러리 사용 보고서 (주가)

1. 사용한 외부 라이브러리 및 버전
  라이브러리명: openpyxl
  설치 버전: 3.1.5
  선택 이유: 파이썬을 이용해서 데이터 시각화를 해보고 싶었고, 엑셀을 자동으로 만들어주는 코드를 짜보고 싶었다.

2. 라이브러리에 관한 이해
   프로그램에서 사용한 `openpyxl`은 파이썬으로 엑셀(`.xlsx`) 파일을 생성, 수정, 제어할 수 있는 외부 라이브러리이다.
   텍스트 데이터를 저장하는 것 뿐만 아니라 엑셀 고유의 기능인 수식 계산과 그래프 그리기 기능을 파이썬 코드로 제어할 수 있는 코드들을 배워서 사용해보았다.
   또 모듈을 통해서 엑셀 시트의 특정 행과 열의 좌표를 데이터 범위로 지정해서 그래프를 그려내는 방법등도 활용해 보았다.

3. 프로그램 기능 설명
 1}'샤페론'의 주가 데이터(주가 변동률, 거래량, 종가)를 바탕으로 엑셀 파일(`shaperon_stock.xlsx`)을 자동 생성해보았다
 2}파이썬 반복문을 이용해서 거래대금을 계산했다
 3}데이터 하단에는 엑셀 내장 함수인 `=AVERAGE()` 수식을 넣어서 평균종가를 확인할 수 있었다
 4}`openpyxl.chart`를 활용하여 일별 종가  추이를 시각적으로 확인할 수 있는 그래프 자동으로 그려서 삽입할 수 있게 만들었다
