외부 라이브러리 사용 보고서

1. 사용한 외부 라이브러리 및 버전
  라이브러리명: openpyxl
  설치 버전: 3.1.5
  선택 이유: 파이썬을 이용해서 데이터 시각화를 해보고 싶었고, 엑셀을 자동으로 만들어주는 코드를 짜보고 싶었습니다

2. 라이브러리에 관한 이해
   프로그램에서 사용한 `openpyxl`은 파이썬으로 엑셀(`.xlsx`) 파일을 생성, 수정, 제어할 수 있는 외부 라이브러리입니다.
   텍스트 데이터를 저장하는 것 뿐만 아니라 엑셀 고유의 기능인 수식 계산과 그래프 그리기 기능을 파이썬 코드로 제어할 수 있는 코드들을 배워서 사용해보았습니다.
   또 모듈을 통해서 엑셀 시트의 특정 행과 열의 좌표를 데이터 범위로 지정해서 그래프를 그려내는 방법등도 활용해 보았습니다.

3. 프로그램 기능 설명
 1}'샤페론'의 주가 데이터(주가 변동률, 거래량, 종가)를 바탕으로 엑셀 파일(`shaperon_stock.xlsx`)을 자동 생성했습니다
 2}파이썬 반복문을 이용해서 거래대금을 계산했습니다
 3}데이터 하단에는 엑셀 내장 함수인 `=AVERAGE()` 수식을 넣어서 평균종가를 확인할 수 있었습니다
 4}`openpyxl.chart`를 활용하여 일별 종가  추이를 시각적으로 확인할 수 있는 그래프 자동으로 그려서 삽입할 수 있게 만들었습니다
